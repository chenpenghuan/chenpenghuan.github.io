# 港区appleId注册

分两步，先创建大陆appleId，再改成港区

创建大陆appleId按照官方指引自行注册即可，下面是appleId中国大陆改港区的流程

1. 第一步，进入浏览器，打开网页版苹果id界面[**https://appleid.apple.com**](https://appleid.apple.com/)；登录你的大陆appleId
2. 进入账户后选择「个人信息」->「国家或地区」国家或地区选香港，付款方式选无，其他信息随便写
